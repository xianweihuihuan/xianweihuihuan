## 关于我

我偏爱那些需要长期积累、也值得长期积累的方向。

主线一直围绕 `Linux`、`操作系统`、`网络编程`、`数据库内核`、`算法设计` 与 `性能优化` 展开。  
对我来说，真正有意思的事情不是把工具用起来，而是把问题想透，把关键路径做出来，把系统行为看清楚。

语言上以 `C / C++` 为核心，也持续用 `Go` 和 `Python` 扩展工程边界。近一阶段，我把注意力继续延伸到 `LLM Inference`、`vLLM`、`KV Cache` 和 `Agent` 相关方向，希望把底层能力自然接到 AI 基础设施上。
