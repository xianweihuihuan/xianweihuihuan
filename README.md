## 关于我

我是一个偏爱底层、系统与长期主义的开发者。

比起把功能堆出来，我更在意把问题做深: 理解它为什么成立，系统怎样运转，瓶颈出现在哪里，优化是否真正有效。  
技术对我的吸引力，不只在于“能做”，更在于“能想清楚、能做出来、也能解释清楚”。

我长期把注意力放在这些方向上:

- `Linux` 与 `System Programming`
- `操作系统` 与计算机基础
- `网络编程` 与服务端实现
- `数据库内核` 与执行机制
- `算法设计` 与复杂问题建模

- `LLM Inference`、`vLLM`、`KV Cache`、`Agent`
